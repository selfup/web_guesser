# web_guesser
Introductory Sinatra app that simulates a number guessing game

# Setup
If you do not have Sinatra or Sinatra Reloader, do the following:

    gem install sinatra
    gem install sinatra-reloader

Now cd into the appropriate directory

    $ ruby web_guesser.rb
  
Go to your browser

    http:localhost:4567

## Enjoy!
